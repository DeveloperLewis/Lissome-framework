# Square, a PHP Framework.

A PHP framework built for simplicity and speed.

## Features

- Lightweight
- User Authentication
- Custom Commandline System
- Phpunit
- Bootstrap

## Installation

- Requires PHP 8, mysql and apache2.
- `git clone https://github.com/DeveloperLewis/basic-php-mvc-framework.git .`
- `composer install`
- `php square env`
- `php square init`
