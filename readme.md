# Square, a PHP Framework.
A framework based on the MVC design pattern. 

## Features
- Lightweight
- User Authentication
- Commandline commands

## Installation
- Requires PHP 8, mysql and apache2.
- `git clone https://github.com/DeveloperLewis/basic-php-mvc-framework.git .`
- `composer install`
- `php square env`
- `php square init`
